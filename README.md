# ICRA2024

This repo contains the f1tenth system made by the F1Tenth 2023 team

## Setup


```
git clone https://github.com/NTU-Autonomous-Racing-Team/ICRA2024.git
cd ./ICRA2024/f1tenth_ws
colcon build
```
